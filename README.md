# QuickCopy
一款快速复制的小工具；目前已支持mac/win/web； 

在线地址:[QuickCopy](http://quickcopy.kongdf.com)

## 技术栈

前端:Electron + Vue
服务端:koa + mysql

* [x] MAC客户端
* [x] Windows客户端
